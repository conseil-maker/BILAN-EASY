## Création de Nouveaux Comptes de Test

Pour créer un nouveau compte de test, utilisez la page de connexion :
1. Aller sur https://bilan-easy.vercel.app/
2. Cliquer sur "Créer un compte"
3. Entrer l'email et le mot de passe
4. Confirmer l'email (vérifier les spams)
